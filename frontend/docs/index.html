isExistingRecipe

Determines whether the input is an existing meal from the meal plan.

isExistingRecipe
Parameters
recipe (object) a data structure of a recipe containing the recipies
Name Description
recipe.name string The name of the input recipe
recipe.ingredientList Array<object> List of ingredients for recipe
Returns
object: list of data structures that represent ingredients If the input exists, the list is populated with each ingredient's properties If the input does not exist, a single empty ingredient is returned.

handleInputChange

This function is called whenever an ingredient field is changed. The function determines what object is being modified and updates the list of ingredient fields to include the input change.

handleInputChange
Parameters

handleAddFields

This function is called when the user clicks the Add button to add another row of ingredient fields. The function pushes an empty ingredientField object onto the ingredientFields list.

handleAddFields

handleRemoveFields

This function is called when the user clicks the Remove button to remove the lowest row of ingredient fields. The function pops an ingredientField object off the ingredientFields list.

handleRemoveFields

handlePopulateIngredients

This function is called when the user modifies the Recipe Name textbox. The ingredientFields data structure is set to the return of the isExistingRecipe function. The recipeName string is set to recipe.name.

handlePopulateIngredients
Parameters
recipe (object) a data structure of a recipe containing the recipies
Name Description
recipe.name string The name of the input recipe
recipe.ingredientList Array<object> List of ingredients for recipe

handleDeleteMeal

This function is called when the user clicks the Delete Meal button to remove the recipe from the weekly meal plan. The removeMeal function is called to locate and remove the recipe from the meal plan. The cancelButtonClicked function is called to close the popup without saving.
